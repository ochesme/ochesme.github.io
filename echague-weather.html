3. Problem:

Lightning strikes 16090 meters away from you. How long does it take the sound of thunder to get to you at 20ºC temperature? *

Note:

   The speed of sound at  20ºC temperature is 343 m/s

   The formula for Speed is speed = distance / time ( s = d / t )

Given Data:

d ( distance ) = 16,090 m

s ( speed ) = 343 m/s

t ( time ) = ? unknown

Solution:

The formula for Speed is speed = distance / time ( s = d / t ), we can the derivation formula which is ( t = d / s ) in order to solve the problem:

t = d / s

= 16,090 m / 343 m/s

= 46.91 s

Answer:

It takes 46.91 seconds in order for you to heat the sound of the thunder
