Problem:

An ocean waves travels at the rate of 20m/s. The Distance between two consecutive crests is 35 meters what is the period of the wave.

Note:

The formula for wave Period is { Period = Wavelength ÷​ velocity ( T = λ / v ) }

Given Data:

λ ( Wavelength ) = 35 m

v ( velocity ) = 20 m/s

T ( Period ) = ? unknown

Solution:

We can use the formula for period which is  { Period = Wavelength ÷​ velocity ( T = λ / v ) }

T = λ / v

  =  30 m / 20 m/s

  = 1.5 s

Answer:

The wave period is equal to 1.5 seconds
