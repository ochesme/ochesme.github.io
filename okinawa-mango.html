This is to check if the difference is possible for the number of yellow mangoes

Via trial and error, if the total number of fruits is 7674, there would be 3880 green mangoes and 3794 yellow mangoes. 7674 = 7674

7674 total mangoes

3888 green mangoes

N=2a + 1

N=7f

N + 1=3(b +1)

N + 1=5(d +1)

N + 1=6(e +1)

N + 1=7f + 1

Take least value of n so that N + 1=120

So 119 is the answer.

11. A three-stringed banjo that originated from Okinawa Japan. Traditionally, snakeskin is used for the body of this instrument.​


Answer:

Sanshin

Explanation:

The Sanshin, meaning three-stringed instrument, is an Okinawan banjo-like instrument, based on the Chinese sanxian, that emerged in Japanese history in the 1700's. Similar to the Japanese Shamisen, the Sanshin has three strings that are stretched over an (artificial) snake skin body, and wooden neck.
